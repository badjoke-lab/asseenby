# AsSeenBy v0.1 — MVP Spec

## Version goal
Ship a static-image-only public MVP with browser-side processing and a strong visual identity.

## Included in v0.1
- image upload
- sample image loading
- slider / split / side-by-side comparison
- category switching
- mode switching
- strength slider
- off-white editorial interface
- right-side control rail
- human / animal / reference grouping

## Human modes
### Strong
- Protan-like
- Deutan-like
- Tritan-like
- Blur
- Low Contrast
- Cataract-like
- Tunnel Vision
- Central Loss

### Estimated
- Night / Low Light
- Fatigue-like
- Dry-eye-like

## Animal modes
- Dog
- Cat
- Bee
- Bird-like

## Reference modes
- Age Profile
- Sex-difference Profile

## Not included in v0.1
- video processing
- live camera mode
- user accounts
- saved sessions
- server-side transformation
- share links
- API
- diagnostic output

## Functional priorities
### P0
- core compare experience
- upload and sample image flow
- mode switching without page reload
- first-pass transformation reliability

### P1
- visual polish
- documentation pages
- cleaner error states
- stronger animal/reference explanation

## Public positioning
AsSeenBy is a research-based comparison tool, not a medical or diagnostic product.
